# VitalIQ - Enterprise SaaS Analytics Dashboard

## Overview

VitalIQ is a responsive Enterprise SaaS Analytics Dashboard designed for B2B businesses to monitor KPIs, revenue, customer insights, and operational performance.

The project was designed in **Figma** as part of a UI/UX Design course.

---

## Features

- Admin Dashboard
- Manager Dashboard
- Analyst Dashboard
- Responsive Mobile Design
- Design System
- Interactive Prototype
- KPI Cards
- Revenue Analytics
- Traffic Source Visualization
- Recent Transactions
- Activity Feed

---

## Tools Used

- Figma
- Iconify
- Charts Plugin
